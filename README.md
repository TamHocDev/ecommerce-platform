# E-commerce Platform

## Overview
This repository contains the code for an e-commerce platform built with Node.js, Next.js, and PostgreSQL.

## Features
- User authentication
- Product catalog
- Cart functionality
- Order management
- Payment integration
- Admin panel

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/TamHocDev/ecommerce-platform.git
   cd ecommerce-platform
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up the database:
   1. Install PostgreSQL.
   2. Create a database.
   3. Update the database configuration. See `config/db.js`.
4. Start the server:
   ```bash
   npm run dev
   ```

# User Journey Documentation

## Customer Personas
### Persona 1: New Customer  
- **Name:** John Doe  
- **Age:** 28  
- **Occupation:** Software Engineer  
- **Goals:** Quick and easy access to products, compare prices.

### Persona 2: Regular Customer  
- **Name:** Jane Smith  
- **Age:** 35  
- **Occupation:** Marketing Manager  
- **Goals:** Frequent purchases, loyalty rewards.

## Guest vs Logged-in Journey
- **Guest Journey:**  
 1. Browsing Products  
 2. Adding items to Cart  
 3. Checkout (with email capture)  
 4. Confirmation Email  

- **Logged-in Journey:**  
 1. Account Sign-In  
 2. Personalized Recommendations  
 3. Adding items to Cart  
 4. Simplified Checkout  
 5. Order Confirmation and Tracking  

## Order Status Flow
1. **Order Placed**  
2. **Processing**  
3. **Shipped**  
4. **Out for Delivery**  
5. **Delivered**  
6. **Cancelled**  

## Key Performance Indicators (KPIs)
- Conversion Rate  
- Average Order Value  
- Customer Retention Rate  
- Cart Abandonment Rate  
- Customer Satisfaction Score
